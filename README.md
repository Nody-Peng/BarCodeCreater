
# BarCode條碼自動生成器

這是一個可以自動編寫條碼，並且將結果儲存於.doc檔案內的程式。


## 使用方法

    1. 請先下載【BarCode_Creater.exe】
    2. 直接執行接著就會看到以下畫面
    ** 目前.exe檔案無法使用故若需要使用此程式需在本地端執行且需要下載相對應套件 **
